You mission is to convert some function name (string) from Python format (for example "my_function_name") into CamelCase ("MyFunctionName"), where first char of every word is in uppercase and all words are concatenated in one.

Input: Function name as string

Output: The same string, but in CamelCase

Example:

{% if interpreter.slug == "js-node" %} 
toCamelCase("my_function_name") == "MyFunctionName"
toCamelCase("i_phone") == "IPhone"
toCamelCase("this_function_is_empty") == "ThisFunctionIsEmpty"
toCamelCase("name") == "Name"
{% else %} 
to_CamelCase("my_function_name") == "MyFunctionName"

to_CamelCase("name") == "Name"
{% endif %}

How it is used: To use name of the functions in that style which are used in the language (Python, JavaScript, etc.).

Precondition:
0 < len(string) <= 100
no numbers in the input
